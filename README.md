debate-planner
A Claude Code skill that turns planning sessions into structured debates — so you build the right thing instead of the complete thing.
What it does
When you're planning a product, feature, or system, this skill makes Claude Code act like a sharp co-founder rather than a yes-machine. It will:

Research first — checks what already exists, known failure patterns, and underestimated complexity (max 3 searches, no token waste)
Audit your features — labels each one as CORE ✅, NICE ⚠️, BLOAT ❌, or REFRAME 🔁
Cut scope ruthlessly — tells you directly when a feature doesn't serve your stated goal
Make one concrete recommendation — not a menu of options, a real call with Phase 1 / Phase 2 scope
When it triggers
Automatically activates when you say things like:

"I'm thinking about building X"
"Help me plan / scope / architect this"
"What do you think of this idea"
"Let's brainstorm the features for…"
